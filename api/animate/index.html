Animate

The animate action allows you to animate cgs. The animation can be a tween of the sprite properties, or a spritesheet animation (coming soon). A time should be specified in milliseconds, that will be how long the tween will take to complete.

    
    - animate cgs_id:
      time: number
      [alpha: number]
      [zoom: number]
      [position: coordinate]
      [angle: number]
